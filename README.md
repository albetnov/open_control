Welcome to my personal Flutter AI setup!

This scaffold drives an AI agent through preparing a new Flutter project: architecture, conventions, and skills, all pre-baked.

## How to use

1. Create a new Flutter project
2. Copy everything in this repo into your project's root
3. Say "@AGENTS.md adopt"
4. That's it

## Prerequites
- `npx` for skills
- dart mcp installed to your agents running this scaffold
