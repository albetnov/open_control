# OpenControl

OpenControl is a Flutter application that allows users to control OBS (Open Broadcaster Software) remotely. It provides a user-friendly interface to manage scenes, sources, and other settings in OBS.

## Features

- Connect to OBS via WebSocket
- Control scenes and sources
- Manage settings and configurations

> Project is currently in development and may not be fully functional. Contributions are welcome!

## Contributing

Setting up the development environment:

1. Clone the repository
2. Run `flutter pub get` to install dependencies
3. Install OBS and set up the WebSocket server
4. Run the application using `flutter run`
5. Make changes and submit pull requests for review
